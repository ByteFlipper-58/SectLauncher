## Welcome to SectLauncher 26.0! 🚀

We are excited to announce our first major release. The launcher has been completely reimagined and rebuilt from scratch using **Rust** and **Tauri 2** to deliver maximum speed, security, and a modern user experience for all Minecraft fans. SectLauncher had a predecessor built with *Compose Multiplatform*, which for various reasons never reached a public release, but today we are proud to present a completely new product.

---

### 📦 Library & Mods (Powerful Features)
No more manual file downloading. We have integrated **Modrinth** directly into the launcher:
* **Unified Search**: Millions of mods, resource packs, and shaders available in a single window.
* **Smart Filters**: Filter by loaders (Fabric/Forge), game versions, and content categories.
* **One-Click Install**: The launcher automatically downloads the correct version and all required dependencies.
* **Content Management**: Easily toggle mods on/off, check their size, and track modification dates right from the list.

### 🍱 Modpack Catalog
Want to play pre-configured modpacks? The **Modpacks** tab is dedicated to this:
* **Instant Import**: Find any modpack on Modrinth and install it as a new instance in seconds.
* **.mrpack Support**: Import your favorite packs from local files with ease.

### ⚙️ Flexible Instance Management
* **Vanilla, Forge, and Fabric Support**: The launcher automatically prepares the environment for any of these platforms.
* **Absolute Control**: Configure Java Runtime, memory allocation (RAM), JVM arguments, and screen resolution for each instance individually.
* **Repair Feature**: If game files are corrupted, the "Repair" function will verify integrity and redownload the correct versions.

### 🛡️ Accounts & Security
* **Microsoft Accounts**: Modern login via Xbox Live (Device Code Flow) — secure and convenient.
* **Offline Mode**: Play with your own nickname without an internet connection.
* **Multi-Account Support**: Switch between profiles in two clicks without re-logging.

### 🎨 Premium Design & Aesthetics
* **Minecraft Style**: Interface inspired by the game's aesthetics, featuring sharp lines and modern animations.
* **Color Magic (Vibrant)**: Customize everything! Change backgrounds, create your own palettes, or let the launcher adapt to the colors of your wallpaper.
* **Game Console**: Monitor startup logs in real-time with automatic error highlighting.

### ⚡ New Rust Core
* **Instant Startup**: Powered by Rust, the launcher consumes minimal resources and starts much faster than its predecessors.
* **Auto-Updates**: The launcher automatically checks for and installs updates — you're always on the latest version.
