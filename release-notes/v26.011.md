# SectLauncher v26.1.0

SectLauncher v26.1.0 makes instance creation smarter and simplifies game setup.

## Highlights

- Java is no longer selected globally in settings.
- The launcher now picks Java for each instance during creation.
- Settings now include a cleaner Java manager and new performance profiles.

## What's New

- In **Settings > Game**, you can download Java 8, 17, and 21, add your own Java path, refresh the list, and remove extra runtimes.
- Java versions that are already installed are no longer shown again in the download list.
- Duplicate Java entries are hidden from the runtime list.
- During instance creation, the launcher automatically chooses a suitable Java version for the selected Minecraft version and loader.
- Java compatibility is shown more clearly while setting up an instance.
- A new default performance profile system was added: **Auto**, **Light**, **Balanced**, **Heavy**, and **Custom**.
- In **Auto** mode, the launcher recommends memory and JVM arguments during instance creation based on the selected version and loader.
- If you manually change memory or JVM arguments for an instance, they stay as that instance's own override.
- Loader version selection became more consistent for supported loaders during creation and repair.

## Upgrade Notes

- Existing instances keep their own Java and launch settings.
- If the required Java version is missing, SectLauncher can still download it automatically during installation.
