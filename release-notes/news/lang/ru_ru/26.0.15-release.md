## SectLauncher 26.0.15-release — Проще настройка, удобнее навигация и стабильнее запуск

### ☕ Меньше ручной настройки Java
* SectLauncher теперь точнее понимает, какая версия Java нужна разным версиям Minecraft.
* При создании или восстановлении инстанса лаунчер лучше подбирает подходящую Java автоматически.
* Это должно уменьшить случаи, когда игра не запускается из-за неправильно выбранной Java.
* Уже существующие инстансы сохраняют свои настройки запуска: Java, RAM, JVM-аргументы, разрешение и полноэкранный режим не сбрасываются.

### 🧭 Быстрый доступ к новым версиям Minecraft
* Добавлены специальные пункты **Последний релиз** и **Последний снапшот**.
* Их можно использовать, если хочется быстро запускать самую свежую стабильную версию Minecraft или последний snapshot.

### 🪟 Больше контроля над боковой панелью
* Добавлена настройка, которая позволяет скрывать боковую панель на выбранных страницах.
* Это удобно, если хочется больше места на страницах Библиотеки, Миров, Настроек или Что нового.

### 🚀 Более надёжный запуск
* Окно запуска и сплеш-скрин теперь должны работать стабильнее.
* Лаунчер аккуратнее обрабатывает проверку обновлений, EULA, загрузку темы и состояние запуска.
