## SectLauncher 26.0.5 — Что нового

### 🌍 Миры и быстрый запуск
* Добавлен полноценный **Worlds Manager**: поиск, сортировка, фильтры по инстансам, загрузчикам и режимам игры, а также запуск мира в один клик.
* Миры можно **группировать по инстансам**, быстро находить по названию и просматривать служебную информацию вроде даты последнего запуска и размера.
* Для миров доступны **переименование, перенос между инстансами, резервные копии, восстановление и удаление** прямо из лаунчера.
* На главной странице появился **Quick Play** в стиле лаунчера для **миров, серверов и Realms** с закреплением элементов, всплывающими карточками и настройкой режима иконок.
* Быстрый запуск умеет показывать либо **иконки загрузчиков**, либо **иконки самих миров**, а предпочтения сохраняются в настройках.

### ⚙️ Настройки и интерфейс
* Полностью обновлена **страница настроек**: чище навигация, лучше структура разделов и плавные переходы между категориями.
* Разделы **General / Game / Appearance / System / About** теперь ощущаются как единая цельная панель, а не набор разрозненных экранов.
* Улучшены анимации и состояние интерфейса в оболочке лаунчера, включая переходы между страницами и поведение skeleton/loading-состояний.
* Все модальные окна теперь стабильно остаются **поверх интерфейса** и аккуратно анимируются при открытии и закрытии, без конфликтов с сайдбаром и другими слоями UI.

### 🧩 Лаунчер и совместимость
* Восстановлена работа **offline multiplayer** и **Realms** на разных версиях Minecraft.
* Для offline-аккаунтов UUID теперь создаётся сразу при добавлении аккаунта, что делает запуск стабильнее и убирает временные расхождения в профилях.
* Исправлена проверка **Repair** для инстансов через **SHA1-валидацию**, поэтому восстановление надёжнее определяет повреждённые или отсутствующие файлы.
* Улучшена обработка зависимостей модов, карточек модов и навигации назад внутри лаунчера.

### 💻 Стабильность на платформах
* Исправлена кроссплатформенная обработка **runtime/launcher**, особенно на **macOS**.
* Усилена стабильность запуска окна и работы локальных ассетов на **macOS/Linux**, включая splash/startup-сценарии.
* На Windows оптимизированы релизные сборки, улучшена упаковка приложения и исправлена загрузка кешированных медиа.
* Windows-установщик теперь умеет автоматически устанавливать **WebView2**, сохраняя лёгкий размер инсталлятора и упрощая первый запуск на чистой системе.

### ⚙️ Под капотом
* Проведён крупный рефакторинг фронтенда и Tauri-бэкенда с более строгим разделением слоёв.
* Фронтенд перестроен вокруг структуры **app / features / shared**, а бэкенд Tauri разделён на более строгие **application / domain / infrastructure** слои.
* Выделен платформенный infrastructure layer, чтобы упростить дальнейшую поддержку Windows, Linux и macOS.
