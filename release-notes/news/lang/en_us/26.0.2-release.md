## SectLauncher 26.0.2 — What's New

### 🧩 Loaders & Modpacks
* Added **NeoForge** and **QuiltMC** support: create instances, import `.mrpack`, and install mods/modpacks for these loaders directly in the launcher.

### 📦 .mrpack Export
* Export your own modpack as **.mrpack**.
* Configurable fields: **icon, name, description, version**.
* Fine‑grained **Overrides** to control what goes into the archive.

### 🔐 Accounts & Security
* **Microsoft token refresh** before game launch: the launcher refreshes the `refresh_token`, fetches a new Minecraft `access_token`, and persists the updated profile.
* **Encrypted account storage**: data is now stored in `accounts.dat` using **AES‑256‑GCM** (`SLv1` format).
* **Key stored in system keychain**: Windows Credential Manager / macOS Keychain / Secret Service (Linux).
* If keyring is unavailable, a plaintext fallback is used with a warning in logs.

### 🧠 UI/UX
* Add Account is now a **modal window**.
* All modals are forced to the top layer — the sidebar no longer overlaps them.

### 🛠️ Bug Fixes
* Miscellaneous fixes and stability improvements.

### ⚙️ Under the Hood
* Major core refactor completed.
* A second frontend refactor is planned to optimize performance and improve UI responsiveness.
