## SectLauncher 26.0.15-release — Easier setup, better navigation, and a more stable start

### ☕ Less manual Java setup
* SectLauncher now understands which Java version is needed for different Minecraft versions more accurately.
* When you create or repair an instance, the launcher has a better chance of choosing the correct Java automatically.
* This should reduce cases where the game does not start because the wrong Java version was selected.
* Your existing instances keep their own launch settings, so this update should not overwrite your custom Java, RAM, resolution, or fullscreen preferences.

### 🧭 Easier access to the newest Minecraft versions
* Added special **Latest Release** and **Latest Snapshot** entries.
* You can use them if you want a quick way to play the newest stable Minecraft version or the newest snapshot.
* The home screen now handles these entries more cleanly.
* Fixed a problem where the launcher could repeatedly refresh the selected home instance in some cases.

### 🪟 More control over the sidebar
* Added a setting that lets you hide the sidebar on selected pages.
* This is useful if you want more space for pages like Mods, Worlds, Settings, or News.
* When the sidebar is hidden, the page now shows a back button so you can still return easily.
* When the sidebar is visible, the extra back button is hidden to keep the interface cleaner.

### 🚀 More reliable startup
* The startup window and splash screen should behave more reliably.
* The launcher now handles update checks, EULA checks, theme loading, and startup state more smoothly.
* This should make opening the launcher feel less glitchy, especially right after an update.

### 🖼 Better changelogs, images, and embedded content
* Changelog pages and Markdown content now render media more reliably.
* Remote images have better fallback behavior if something fails to load.
* Embedded videos and external content should feel more consistent inside the launcher.

### 🧱 Cleanup for future updates
* A lot of internal launcher code was cleaned up and split into smaller parts.
* This does not add a flashy button by itself, but it makes future fixes and features easier to ship.
* Quick Play, mod installation, mod imports, mod details, worlds, and settings were all part of this cleanup.
* Some unused old code was removed, which helps keep the launcher easier to maintain.
