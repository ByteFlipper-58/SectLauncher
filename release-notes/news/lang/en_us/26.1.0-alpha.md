### Welcome to SectLauncher!

This is the first public alpha version of the launcher. We rebuilt it from scratch on **Rust** and **Tauri 2** to deliver faster startup, a more responsive interface, and a more reliable core.

### What's already in 26.1.0-alpha

* **New Rust core**: async workflows powered by Tokio for settings, accounts, and game launch.
* **Instance management**: create, edit, import, export, and repair game profiles.
* **Vanilla, Forge, and Fabric setup**: prepare builds directly from the launcher.
* **Microsoft and Offline accounts**: Device Code Flow login plus quick offline access.
* **Multi-account support**: switch between profiles without restarting the client.
* **Advanced settings**:
  * Java path configuration;
  * flexible memory allocation;
  * JVM arguments;
  * window resolution and fullscreen mode.
* **Game console**: live logs, visible error states, and direct launch diagnostics.
* **New launcher shell**: a Minecraft-inspired interface with themes and localization support.
